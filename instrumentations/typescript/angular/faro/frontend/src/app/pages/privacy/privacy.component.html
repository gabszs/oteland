Política de Privacidade

Introdução

Sua privacidade é importante para nós. Esta política de privacidade explica como coletamos, usamos e protegemos seus dados.

Coleta de Dados

Coletamos informações que você nos fornece voluntariamente através do uso do aplicativo.

Uso de Dados

Os dados coletados são usados apenas para melhorar a experiência do usuário e fornecer os serviços solicitados.

Proteção de Dados

Implementamos medidas de segurança para proteger seus dados contra acesso não autorizado.

Contato

Se tiver dúvidas sobre nossa política de privacidade, entre em contato conosco.
